# SecurePass-AI
SecurePass AI is an AI-powered password security platform that checks password strength, generates strong passwords, and helps users improve their online security.

## Features
- Password Strength Checker
- Password Generator
- AI Password Suggestions
- Login / Signup
- Admin Dashboard
- Profile Page
- Saved Passwords

## Tech Stack
- HTML
- CSS
- JavaScript
- Python Flask
- MySQL
- OpenAI API

## Note
AI features require OpenAI API key in backend/.env
